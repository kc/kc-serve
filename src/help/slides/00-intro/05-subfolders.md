Group modules into sub folders*:

```bash
/css
/img
/slides
  + /00-first module
  |  + 00-first subject.md
  |  + 10-second subject.md
  |  + 11-some nice image.png
  |  + 20-third subject.md
  + /10-second module
  |  + 00-first other subject.md
  |  + 10-second other subject.md
  + /20-third module
     + 00-et cetera.md
     + 01-images are also slides.gif
```

<small>* Do not combine with separators</small>
